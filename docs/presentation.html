10 Why Hybrid Search (BM25 + Vector)?

🔍 The Problem: Each approach has blind spots

📄 BM25 Full-Text Only

Good at: Exact keyword matching, rare/technical terms like "BMW 520i 2022"

Misses: Synonyms, paraphrases, semantic meaning — "sedan" won't match "executive saloon"

Score: Keyword overlap only, no understanding of intent

🧠 Vector Embedding Only

Good at: Semantic matching, synonyms, conceptual similarity — "family car" matches "7-seater SUV"

Misses: Exact/tail terms — "BMW 520i 2022" might rank a similar Mercedes higher

Score: Cosine similarity, can drift on rare terms

✅ Hybrid (BM25 + Vector + RRF) — Best of Both
Query → BM25 score + Vector similarity → RRF fusion → Final ranked results
🎯
Exact matches
boosted by BM25
🌊
Semantic matches
caught by vectors
🔀
RRF merges both
no tuning needed
📈
Higher recall
without precision loss

🧪 Real-world example: Searching "luxury German sedan" via BM25 misses "Mercedes E-Class" listings that don't contain those keywords — but vector search catches the semantic match

23 Key Design Decisions

🔎 Hybrid BM25 + Vector Search

Decision: RRF fusion of BM25 full-text + vector cosine search

Alternative considered: Pure vector search with all-MiniLM-L6-v2

Why this won: Pure vector search missed exact keyword matches (e.g., model codes like "520i"). BM25 handles precise term matching while vector captures semantic intent. RRF fusion gives best of both worlds with automatic quality evaluation and broader retry on low scores.

🧠 3-Tier LLM Fallback

Decision: Groq (fast) → Groq (powerful) → OpenRouter → Groq fallback

Alternative considered: Single-provider (Groq only)

Why this won: Groq provides low latency but limited model selection. OpenRouter adds redundancy and access to specialized models (vision, reasoning). The 3-tier chain ensures 99.9% uptime — if Groq is down, OpenRouter takes over, with a final Groq fast model as last resort.

🛡️ Hallucination Guard Layer

Decision: 5-check validation before returning comparison results

Alternative considered: No validation layer (trust LLM output directly)

Why this won: LLMs confidently fabricate car specifications and pricing. The 5-check system (claim trace-back, accident claim validation, mechanical fault restrictions, confidence consistency, winner validity) catches hallucinations before they reach users, critical for a marketplace where inaccurate data could lead to poor purchase decisions.
